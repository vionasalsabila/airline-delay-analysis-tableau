# Airline Delay Analysis Dashboard

## Description
This project presents an interactive Tableau dashboard analyzing airline delay causes across the United States, including carrier delays, weather delays, and other contributing factors.

## Tools
- Tableau

## Key Insights
- Carrier delays contribute the largest portion of total delays
- Delay patterns vary across airlines, with some carriers showing higher delay frequencies
- Geographic analysis shows delay concentration in specific regions
- Delay trends fluctuate over time, indicating possible seasonal or operational factors
